# 🎙️ Голосовой ToDo-менеджер

Приложение для управления задачами с поддержкой голосовых команд. Современный дизайн в стиле Glassmorphism, написан на Vue 3 с TypeScript и Pinia.

![Версия](https://img.shields.io/badge/version-1.0.0-blue)
![Vue](https://img.shields.io/badge/Vue-3.3-green)
![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue)
![Pinia](https://img.shields.io/badge/Pinia-2.1-yellow)
![Лицензия](https://img.shields.io/badge/license-MIT-green)

## ✨ Особенности

- 🎤 **Голосовое управление** - полный контроль задачами голосом
- 🎨 **Glassmorphism дизайн** - современный стиль с эффектом стекла
- 📱 **Адаптивный интерфейс** - отлично работает на всех устройствах
- 💾 **Локальное хранилище** - задачи сохраняются в браузере
- 🎯 **Управление задачами** - добавление, выбор, выполнение, удаление
- 📊 **Статистика** - отслеживайте прогресс выполнения
- ⌨️ **Горячие клавиши** - быстрая работа с клавиатуры

```bash
